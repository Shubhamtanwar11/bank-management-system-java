# Bank Management System

This project is a Bank Management System developed using Java, JDBC, and SQL.  
It is a console-based application that performs basic banking operations and stores data in a relational database.

---

## Project Functionality
- Create customer accounts
- Deposit money
- Withdraw money
- Transfer money between accounts
- View transaction details

---

## Tools and Technologies
- Java
- JDBC
- SQL (MySQL or similar database)

---

## Working of the Project
The application is written in Java and uses JDBC to connect to the database.  
All customer and account information is stored in SQL tables.  
Transactions such as deposits, withdrawals, and transfers update the database accordingly.

For fund transfers, database transactions are used to ensure data consistency.

---

## Database Tables
- customers
- accounts
- transactions

---

## Steps to Run
1. Create the required database tables
2. Update database credentials in the code
3. Compile and run the Java program
4. Use the console menu to perform operations

---

## Purpose of the Project
This project was created for learning Java, JDBC, and SQL and to understand how database-based applications work.
